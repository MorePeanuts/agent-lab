# Chatbot Project

A research-oriented chatbot project built with LangGraph, designed to explore and experiment with modern chatbot architectures and capabilities.

## Project Overview

This project aims to build a feature-rich chatbot using LangGraph as the core framework. The focus is on researching and implementing advanced chatbot features including state management, memory persistence, tool integration, and multi-modal interactions. The project will serve as a learning platform and testbed for chatbot design patterns and best practices.

## Features Roadmap

- [ ] Basic conversational interface
- [ ] Short-term memory management (context window)
- [ ] Long-term memory persistence
- [ ] Conversation history tracking

### Dialogue Rewind & State Editing

- [ ] State inspection at any conversation point
